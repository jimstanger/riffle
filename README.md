Riffle is a very simple trivia game with questions and answers read from a CSV file. It's meant for anyone that wants a drop dead simple way to hold an educational trivia game without having to fiddle with a slide deck, code, servers, or third-party platforms. Vibe coded with Google Gemini Code Assist. 

In riparian ecology, a riffle is a shallow, fast-flowing section of a stream or river characterized by turbulent water moving over a rocky or gravelly bottom.
